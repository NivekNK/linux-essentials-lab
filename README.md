# Linux Essentials Lab

Repositorio de apoyo para la charla "Supervivencia en la Terminal".

## Requisitos
* Acceso a una terminal Linux (Bash).
* Git instalado.

## Inicio
Para comenzar el laboratorio, sigue las instrucciones de la presentación:
1. Clonar este repositorio.
2. Otorgar permisos al archivo `setup.sh`.
3. Ejecutar `./setup.sh` para generar los archivos de práctica.
